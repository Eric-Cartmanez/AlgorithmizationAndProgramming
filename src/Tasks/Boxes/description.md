# Задание: Коробки

В почтовом отделении работник должен упаковать коробку посетителя размером a1 x b1 x c1 в почтовый ящик размером a2 x b2 x c2. Все размеры вещественного типа и больше нуля.

Напишите программу, которая определяет, можно ли это сделать и выводит сообщение yes или no. Ввод данных происходит в порядке: a1, b1, c1, a2, b2, c2.

Используйте оператор условия.
