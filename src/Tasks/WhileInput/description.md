# While input
(Одна из возможных задач в "Задание: Цикл While")

Напишите программу которая осуществляет ввод целых чисел от пользователя в цикле. 0 является признаком выхода. Программа вычисляет и отображает на экране P/S, где P - произведение всех введенных чисел,а S - их сумма.

Проверку вводимых значений на корректность делать не обязательно.
