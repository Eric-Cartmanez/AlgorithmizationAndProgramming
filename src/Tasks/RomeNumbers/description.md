# Задание: Римские цифры (доп)

На вход подается целое число в интервале [1; 100]. Программа должна вывести это число в формате римских цифр.

Буквы должны быть заглавные. Пример:

5  --> V

9  -->  IX

54  --> LIV

99  -->  XCIX

>Разрешено использовать математические операции, оператор условия и оператор выбора.
